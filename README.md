# Heroes of NFT - Nft Smart Contracts

## Installation
```sh
npm i
```

## Deployment
### Avash
```sh
npm run avash:deploy
```

### Fuji Testnet
```sh
npm run fuji:deploy
```
```sh
npm run fuji:export
```

### Avalanche Mainnet
```sh
npm run avalanche:deploy
```
```sh
npm run avalanche:export
```
